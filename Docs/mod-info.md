# Mod Info
The asset located in Assets/ModInfo.asset containing following details of your mod.

<p align="center"><img src="Resources/mod-info.png" alt="Alt Text"></p>

### Code Name
The name of the folder where your mod will be stored. This should be unique name containing only English characters.

### Display Name
The human-readable name of the mod, which is shown in user interfaces. 
We may add localization to it later

### Version
The version of the mod. Follow semantic versioning (e.g., `1.0`) if applicable.

### Author
The name of the mod's creator or team.

### Tags
An array of descriptive tags associated with the mod. These can be used for categorization or filtering in mod managers or tools.
<br/>Examples: visuals, balance, funny
